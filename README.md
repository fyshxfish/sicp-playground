# sicp-playground

## Scheme Enviroment Set-up

MacOS:

```bash
brew install mit-scheme
```

Ubuntu:

```bash
sudo apt-get update 
sudo apt-get install mit-scheme
```

## REPL Basics

Enter the scheme repl:

```bash
mit-scheme
```

Load a scheme file in repl:

```
1 ] => (load "path/to/file")
```

Clear the screen: 

cmd+k (MacOS)
